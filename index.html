🎮 MonsterAR 付費驗證系統

簡化版API系統，透過修改單一數字來控制所有用戶的付費狀態

📋 狀態代碼說明

statusCode 對應行為 說明
1 已付費 正常使用，不做任何動作
2 未付費 顯示浮水印
3 長期拖欠 彈出提示並強制退出
4 網路異常 顯示網路錯誤提示

🔗 API 端點

HTML API: api.html
JSON API: payment-config.json
Unity 請求: api.html?api=true

⚡ 快速操作

查看當前狀態 GitHub 編輯

🔧 維護方式

  1. 編輯 payment-config.json 或 api.html
  2. 修改 statusCode 數值 (1-4)
  3. Git commit + push
  4. 1-2分鐘後生效
